### 【第一节】三个正式案例：AIGC如何改变内容生产（15分钟）

**教师话术**：

> 前两次课我们解决了"给谁看"和"讲什么故事"的问题。
> 从今天开始进入第二单元：**怎么高效地把内容做出来**。
> 先看三个真实的商业案例，看看AIGC在行业里已经走到了哪一步。

#### 案例一：可口可乐 "Create Real Magic"（2023）

**数据**：
- 全球首个将 GPT-4 + DALL-E 结合的品牌营销平台
- 17个国家的用户参与，产出超过 **12万件** AI生成作品
- 优胜作品登上纽约时代广场和伦敦皮卡迪利广场的 20 块数字广告牌
- 圣诞季延伸活动：3周内超过 **100万用户** 在43个市场、26种语言中与AI圣诞老人互动

**教师引导分析**：

> 1. **不是技术展示，是商业闭环**：可口可乐没有自己做12万张图，而是让用户用AI工具+品牌素材自己创作。成本极低，传播极广。
> 2. **品牌资产 + AI = 可控创作**：用户只能调用可口可乐授权的视觉元素（弧形瓶、北极熊、圣诞老人），所以无论怎么生成，品牌调性都不会跑偏。
> 3. **与我们课程的关联**：你们做丝路文化海报，也需要先确定"哪些视觉元素是你IP的品牌资产"，再用AI生成。

**板书关键词**：品牌资产约束 / 用户共创 / 12万件作品 / 商业闭环

#### 案例二：《Cosmopolitan》杂志首个AI封面（2022年6月）

**数据**：
- 人类历史上第一个由AI生成的主流杂志封面
- 使用 DALL-E 2，由数字艺术家 Karen X. Cheng 与杂志编辑团队协作完成
- 单张图片生成只用了 **20秒**，但前期提示词打磨 **超过100小时**

**最终提示词**：

```
wide-angle shot from below of a female astronaut
with an athletic feminine body walking with swagger
toward the camera on Mars in an infinite universe,
synthwave digital art
```

**教师引导分析**：

> 1. **20秒 vs 100小时**：AI出图只要20秒，但提示词的调试花了100多个小时。这说明什么？——**提示词工程才是核心技能**，不是点"生成"按钮。
> 2. **提示词的结构**：这个提示词不是随便写的，它包含了镜头角度（wide-angle shot from below）、主体描述（female astronaut）、动态姿态（walking with swagger）、场景（Mars）、艺术风格（synthwave digital art）。这就是我们接下来要学的"结构化提示词"。
> 3. **Karen Cheng 原话**："这不是输入几个词就能神奇地得到完美图片。过程中有大量的人类判断和决策。"

**板书关键词**：20秒出图 / 100小时调试 / 结构化提示词 / 人类判断不可替代

#### 案例三：数字敦煌——AI驱动的文化遗产保护与传播

**数据**：
- 莫高窟现存 735 个洞窟、**4.5万平方米** 壁画
- 腾讯与敦煌研究院合作，构建了 **30万张** 病害标注数据库，覆盖23种典型壁画病害
- AI病害识别准确率达 **92%**，分割误差小于0.5毫米
- 传统人工完成10平方米壁画病害调查需96小时，AI节省 **70%** 人力
- "数字敦煌"资源库2024年访问量突破 **2300万次**，覆盖全球78个国家
- 全球50万网友参与了第220窟修复方案投票

**教师引导分析**：

> 1. **AI不只是"生成"，还能"修复"**：敦煌团队用AI分析唐代壁画的色彩数据，还原了被氧化变色的原始色调。这是AI在文化遗产领域的真正价值。
> 2. **与你们期末作品的关联**：你们手里有CCTV级的4K丝路纪录片素材，其中很多历史影像画质不够理想。后面我们会学用AI做画质增强和修复。
> 3. **从专业工具到公众参与**：50万人投票决定修复方案，这就是"互动性"的力量。你们的独立站也要有这种参与感。

**板书关键词**：30万标注数据 / 92%准确率 / 修复+再创作 / 公众参与

### 【第二节】提示词工程：从"抽卡"到"工业级"（20分钟）

#### 2.1 为什么你的AI出图总是"差一点"

**教师话术**：

> 很多同学可能已经用过AI画图工具，体验往往是这样的：
> 输入"敦煌壁画风格的海报"，出来四张图，一张还行，三张离谱。
> 然后你点"重新生成"，又来四张，还是随机的。
> 这种玩法叫"抽卡"。运气好抽到一张能用的，运气不好白费时间。
>
> 专业的做法不是抽卡，是**工程化**——用结构化的提示词，让AI每次生成的结果都在你预期的范围内。

#### 2.2 提示词六要素框架

**板书/PPT**：

```
结构化提示词 = 六要素组合

┌──────┬────────────────────────────────────┐
│ 主体 │ 画面的核心对象是什么                │
│      │ 例：一位丝路商人牵着骆驼            │
├──────┼────────────────────────────────────┤
│ 场景 │ 在哪里、什么环境                    │
│      │ 例：塔克拉玛干沙漠日落时分          │
├──────┼────────────────────────────────────┤
│ 风格 │ 什么艺术风格                        │
│      │ 例：敦煌壁画风格 / 赛博朋克 / 水墨  │
├──────┼────────────────────────────────────┤
│ 光影 │ 什么样的光照和氛围                  │
│      │ 例：金色逆光、体积光、暖色调        │
├──────┼────────────────────────────────────┤
│ 构图 │ 镜头角度和画面布局                  │
│      │ 例：低角度仰拍、广角、浅景深        │
├──────┼────────────────────────────────────┤
│ 品质 │ 技术参数和排除项                    │
│      │ 例：高清、4K、电影质感、无文字      │
└──────┴────────────────────────────────────┘
```

**教师话术**：

> 记住这六个词：**主体、场景、风格、光影、构图、品质**。
> 写提示词的时候，像填表一样把六个格子填满，出图质量会立刻上一个台阶。

#### 2.3 坏提示词 vs 好提示词：现场对比

**教师演示**（课前准备好两组生成结果截图）：

**坏提示词**：
```
敦煌壁画海报
```
> 结果：AI不知道你要什么主体、什么构图、什么用途，生成的东西模糊、杂乱、不可用。

**好提示词**（六要素版）：
```
【主体】一位身着唐代丝绸长裙的飞天仙女，手持琵琶，飘带飞扬
【场景】敦煌莫高窟藻井图案作为背景，星空与祥云环绕
【风格】敦煌壁画工笔重彩风格，矿物颜料质感，金箔点缀
【光影】柔和的金色光晕从人物背后散射，温暖而神圣
【构图】居中对称构图，人物占画面三分之二，竖版海报比例
【品质】超高清，细节精致，无文字，无水印
```
> 结果：主体明确、风格统一、构图可控，可以直接用于传播物料。

**教师话术**：

> 差距一目了然。六要素不是让你写更多字，是让你**把模糊的想法变成精确的指令**。
> AI不会读心术，你描述得越精确，它给你的结果就越接近你要的。

**板书/PPT**：

【主体词库】

### 【第三节】课堂实操：用即梦生成丝路文化海报（25分钟）

#### 3.1 工具介绍（3分钟）

> 今天我们用的工具叫**即梦**（jimeng.jianying.com），字节跳动出品。
> 选它三个原因：免费、中文友好、不需要翻墙。
> 普通用户每天可以免费生成 **132张** 高清图，够你们折腾了。

**教师演示**（打开即梦网页）：

> 1. 用抖音账号登录
> 2. 点击"AI作图"→"文生图"
> 3. 左边输入提示词，右边选参数（比例、模型版本）
> 4. 点"立即生成"，等几秒出图

**即梦的两个隐藏技能**（重点提醒）：

> - **DeepSeek对话生成提示词**：不知道怎么写提示词？直接跟内置的DeepSeek对话，说"帮我写一个敦煌飞天海报的提示词"，它会一次给你4条，选一条直接用。
> - **反推提示词**：上传任何一张你喜欢的图片，即梦能反向分析出对应的提示词。这是学习提示词最快的方法。

#### 3.2 教师示范（5分钟）

**教师现场演示完整流程**：

**第一步**：确定海报主题

> 我们用第一课讲的"最大公约数"原则。
> 我选一个"匠心"方向：敦煌壁画修复师。

**第二步**：用六要素写提示词

```
一位年迈的敦煌壁画修复师，戴着老花镜，
手持细毛笔正在修复一幅褪色的飞天壁画，
洞窟内部环境，岩壁上可见残存的彩色壁画，
敦煌工笔重彩风格与纪实摄影混合，
暖色烛光从侧面照亮修复师的面部和双手，
中景构图聚焦人物双手与壁画的接触点，
超高清，细腻的皮肤纹理和颜料质感，竖版海报比例
```

**第三步**：设置参数

> - 模型：即梦 4.5
> - 比例：9:16（竖版海报）
> - 生成数量：4张

**第四步**：看结果，选最好的一张

> 4张里选一张最好的。不满意？调提示词重新生成。
> 注意：**不要每次都改很多**，一次只调一个要素，看变化。

#### 3.3 学生实操（15分钟）

**任务**：每人生成至少一张丝路文化主题海报。

**操作步骤**：

> 1. 打开即梦（jimeng.jianying.com），登录
> 2. 结合你们小组的IP方向，确定海报主题
> 3. 用六要素框架写提示词（写不出来就用DeepSeek辅助）
> 4. 生成，看结果，调整提示词，再生成
> 5. 至少迭代2轮，选出你最满意的一张

**教师巡场要点**：

> - 看到学生写"好看的丝路海报"这类模糊提示词，引导他们用六要素拆解
> - 看到结果不理想的，帮学生定位是哪个要素没写清楚
> - 鼓励学生试试"反推提示词"功能：找一张网上喜欢的图，上传反推，看AI怎么描述

#### 3.4 成果快速分享（2分钟）

> 随机请2位同学投屏展示：你的提示词是什么？生成了什么？哪一轮调整让结果明显变好了？

### 【第四节】风格一致性控制与批量生产逻辑（15分钟）

#### 4.1 为什么"一致性"比"好看"更重要

**教师话术**：

> 刚才每个人都生成了一张还不错的海报。
> 但问题来了：你们期末要交的是一**套**视觉资产包，至少5张海报。
> 如果5张海报风格各异，看起来像5个人做的，那就不是作品集，是杂货铺。
>
> 李子柒的视频为什么一眼就能认出来？因为视觉风格高度统一。
> 可口可乐让12万人用AI创作，为什么品牌调性没乱？因为限定了视觉元素库。
>
> **风格一致性 = 专业感。这是从"学生作业"到"工业级产品"的分水岭。**

#### 4.2 三种保持一致性的方法

**板书/PPT**：

```
方法一：提示词模板复用
━━━━━━━━━━━━━━━━━━━━━━━━━
把六要素中的"风格、光影、品质"固定下来，
只替换"主体"和"场景"。

示例模板：
[主体描述]，[场景描述]，
敦煌工笔重彩风格，矿物颜料质感，金箔点缀，
柔和金色光晕，温暖神圣氛围，
超高清，细节精致，无文字，9:16竖版

海报1：飞天仙女 + 藻井背景 + 以上模板
海报2：壁画修复师 + 洞窟内部 + 以上模板
海报3：丝路商队 + 沙漠黄昏 + 以上模板
→ 三张海报主题不同，但视觉语言统一
```

```
方法二：以图生图（垫图）
━━━━━━━━━━━━━━━━━━━━━━━━━
用你最满意的那张图作为参考图，
上传到即梦的"以图生图"模式，
再输入新的主体描述。
AI会参考原图的色调、构图、风格来生成新图。
```

```
方法三：建立你的"视觉风格卡"
━━━━━━━━━━━━━━━━━━━━━━━━━
在动手之前，先定义好一张风格卡：

我们IP的视觉风格定义：
  主色调：___________（如：赭石+金色+靛蓝）
  艺术风格：_________（如：敦煌壁画+现代极简）
  光影基调：_________（如：暖色侧光）
  禁止出现：_________（如：赛博朋克、霓虹色）

每次生成前对照风格卡检查提示词，
偏离了就拉回来。
```

#### 4.3 课堂快速练习（5分钟）

> 用你刚才最满意的那张海报的提示词作为模板，
> 只替换主体和场景，再生成一张。
> 对比两张图：风格是否统一？如果不统一，是哪个要素跑偏了？

### 【第五节】AI历史素材修复与再创作（10分钟）

#### 5.1 从"修旧如旧"到"生成缺失"

**教师话术**：

> 除了从零生成海报，AIGC还有一个对丝路文化特别有价值的能力：**修复和再创作历史素材**。
> 刚才讲的数字敦煌就是例子。你们手里的纪录片素材里也有大量老旧的历史影像。

**展示两组对比**（教师课前准备截图）：

**能力一：画质增强**

> AI超分辨率技术可以把模糊的老照片/低清视频帧变成高清图像。
> 操作方式：在即梦中，生成图片后点击"升级分辨率"，可以放大到2K。
> 也可以用专门的超分工具（如Topaz、waifu2x）处理视频帧。

**能力二：局部重绘**

> 壁画残缺了？AI可以根据周围的纹样和色彩，"补全"缺失的部分。
> 即梦的"修复"功能：用光标选中需要修改的区域，输入描述，AI重新生成该区域。

**能力三：风格迁移**

> 把一张敦煌壁画转换成水彩风格、浮世绘风格、或现代插画风格。
> 操作方式：以图生图，把原始壁画作为参考图，提示词中指定目标风格。

**教师话术**：

> 这三个能力在你们做期末作品时都用得上：
> - 纪录片截帧画质不够？用AI增强。
> - 想在网页里展示壁画从残缺到复原的对比？用局部重绘。
> - 想做一组"古今对话"海报？用风格迁移。
>
> 但有一条红线：**AI修复和再创作必须标注"AI辅助生成"，不能冒充原始文物**。
> 这是学术诚信，也是文化遗产数字化的行业规范。

## 五、教学注意事项

**关于工具选择**：即梦是首选，免费且中文友好。如果个别学生更熟悉其他工具（通义万相、文心一格、Midjourney），允许使用，但提示词框架是通用的。

**关于网络环境**：即梦不需要翻墙，课前确认教室Wi-Fi可用。如果网络不稳定，教师提前准备好演示截图，学生课后在宿舍完成实操。

**关于版权意识**：AI生成的图片目前各国版权政策不同。课堂上强调两点：（1）不要用AI生成的图冒充实拍或原始文物；（2）在作品中标注"AI辅助生成"。

**关于课堂节奏**：第三节实操环节（25分钟）是本课核心，宁可压缩其他环节也要保证学生动手时间。教师巡场时重点帮助写不出提示词的学生，可以引导他们先用即梦内置的DeepSeek对话功能生成初版提示词，再手动优化。
