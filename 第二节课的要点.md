 第二次课设计逻辑

  核心定位：从"认知冲击"过渡到"工具上手"
  第一课建立了叙事模式的认知框架，
  第二课让学生用数据工具和AI完成
  从"感觉对"到"数据对"的跨越。

  ┌───────┬──────────────────────────┐
  │ 目标  │          怎么做          │
  ├───────┼──────────────────────────┤
  │ 真实  │ 拆解李子柒/阿木爷爷等   │
  │ 案例  │ 真实出海案例的数据与策略 │
  ├───────┼──────────────────────────┤
  │ 工具  │ Google Trends+巨量算数   │
  │ 实战  │ 实操+用户画像构建        │
  ├───────┼──────────────────────────┤
  │ 产出  │ 课堂完成IP策划书框架，   │
  │ 落地  │ 课后补完整版             │
  └───────┴──────────────────────────┘

  90分钟结构：
  1. 20min — 真实案例拆解：谁把中国文化讲出去了
  2. 10min — IP价值评估三维模型
  3. 20min — 数据工具实操+用户画像框架
  4. 20min — AI受众模拟进阶（高级Prompt）
  5. 10min — IP策划书框架填写（课堂实操）
  6. 10min — 平台分发策略+课后作业

  设计决定：
  - 用20分钟真实案例打底，延续第一课认知框架
    并用数据证明"叙事模式确实有效"
  - 数据工具选Google Trends+巨量算数，都免费
  - AI Prompt从单角色→多角色对比+结构化输出
  - IP策划书只要求框架，课后打磨，降低压力
